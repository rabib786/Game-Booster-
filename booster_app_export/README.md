# Nexus Booster MVP

This is a lightweight Windows Game & System Booster app built with Python and Eel.

## Prerequisites
You need Python installed on your Windows machine.

## Installation
1. Open your Command Prompt or PowerShell.
2. Install the required Python libraries by running:
   ```bash
   pip install eel psutil
   ```

## How to Run
Because this app needs to kill processes and delete system temporary files, it **must be run with Administrator privileges**.

1. Right-click on your Command Prompt or PowerShell shortcut and select **"Run as Administrator"**.
2. Navigate to the folder where you extracted these files.
3. Run the application:
   ```bash
   python main.py
   ```

A sleek, gamer-aesthetic window will open, allowing you to boost your game performance and clean your system!
